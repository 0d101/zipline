****************************************************
Zipline - A Pythonic Library for Algorithmic Trading
****************************************************

**Zipline** is a library for creating and analyzing quantitative financial
algorithms in Python.  It is the engine that powers backtesting and live
trading for Quantopian_.

   This documentation is still a work in progress.  Many sections may be
   incomplete or innaccurate.
